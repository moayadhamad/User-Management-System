# User-Management-System
## Description
A simple web application built using HTML and PHP with a MySQL database. It allows users to add a name and age, store the data in the database, display all records, and toggle the status between 0 and 1.

## Features
- Add a new user (Name and Age).
- Store data in a MySQL database.
- Display all users in a table.
- Toggle the Status value between 0 and 1.

## Technologies Used
- HTML
- PHP
- MySQL

## Files
- `index.php` – Main page with the form and user table.
- `toggle.php` – Changes the status value.
